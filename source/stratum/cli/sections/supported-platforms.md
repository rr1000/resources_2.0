# Supported Platforms

Since version 2.0.0, the following platforms and architectures are supported by the Catalyze CLI.

| OS | Architecture |
|----|--------------|
| Darwin (Mac OS X) | 64-bit, 32-bit |
| Linux | 64-bit, 32-bit |
| Windows | 64-bit, 32-bit |
